# ArcGraphModel

## develop

### prerequisites

- .NET 6 SDK

### build

#### Linux/macos

- make `build.sh` executable
- run `build.sh`

#### Windows

run `build.cmd`

#### or run the build project directly:

`dotnet run --project ./build/build.fsproj`

### test

#### Linux/macos

- run `build.sh runTests`

#### Windows

run `build.cmd runTests`

#### or run the build project directly:

`dotnet run --project ./build/build.fsproj runTests`
